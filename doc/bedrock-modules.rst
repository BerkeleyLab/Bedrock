Bedrock Modules
===============

DSP
---

In the DSP directory you can find various various digital signal
processing algorithms implemented in platform-independent (portable)
Verilog, and their test benches; modules include DDS, Down-conversion,
Up-conversion, CIC Filters, Low-Pass filters, High-Pass filters, Mixers.
