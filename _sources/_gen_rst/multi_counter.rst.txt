.. _multi_counter:

multi_counter
=============

Description
'''''''''''

|  Single-clock-domain multi-channel counter
|  No restrictions on how often the "inc" port is high.
|  Reads go through dpram, so are delayed one cycle.
|  Reads are passive, so don't need an enable.
|
